## 🧪 List of Experiments

Each experiment is organized in its own folder with relevant source code and documentation.

| No. | Experiment Title |
|-----|------------------|
| **1**  | Create a Node.js server that serves static HTML and CSS files without using Express. |
| **2**  | Build a Node.js server using Express to store form data as JSON and display it via Handlebars. |
| **3**  | Develop a Node.js + Express app that performs CRUD operations on student details using MongoDB. |
| **4**  | Design a simple counter application using AngularJS. |
| **5**  | Build a Todo application using AngularJS with data persistence in a JSON file. |
| **6**  | Implement a user authentication system (Sign-Up/Login) using cookies with MongoDB/MySQL. |
| **7**  | Create and deploy a virtual machine accessible via SSH using a virtual box. |
| **8**  | Containerize and deploy a Node.js ping server using Docker. |
| **9**  | Build an MCQ quiz application with dynamic question fetching and result calculation. |
| **10** | Develop a blog system with admin panel and content storage in MongoDB/MySQL. |
| **11** | Create a calculator app using AngularJS that mimics a mobile calculator interface. |
